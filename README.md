# quick-install
Shell script to install useful developer packages

## Mac
To install included tools, copy the following lines:
```
sudo curl -fsSL https://raw.githubusercontent.com/tmcdo1/quick-install/master/install_mac | /bin/bash
```
or
```
chmod +x install_mac
./install_mac
```

Included installs:
- Homebrew
- Node
- Python 3
  - Flask
- Node.JS
- Docker
- Git
- Visual Studio Code

## WSL
Included installs:
- Python 3
  - Flask
- Docker
- Git
- Visual Studio Code
